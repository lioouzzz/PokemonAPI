# Pokémon API 小網頁

使用 [PokeAPI](https://pokeapi.co/) 打造的互動式前端應用，能搜尋、瀏覽寶可夢資料。

## 功能
- 📄 顯示圖片、編號、類型、持有道具
- 🔊 播放寶可夢叫聲
- ⏪ 分頁功能：每頁顯示 20 筆

## 技術
- HTML / CSS / JavaScript
- Fetch API（串接 PokeAPI）
- Font Awesome 圖示 + Google Fonts 字型
